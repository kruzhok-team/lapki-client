00. Раздел компилятора

Для компилятора потребуется подключение к интернету!

Модуль компилятора служит для преобразование схемы машины состояний в формат выбранной целевой платформы.

Разделы документации о компиляторе:

  1. Импорт схемы из формата игры «Берлога: Защита пасеки» — описывает действие для импорта схемы в IDE из игры «Берлога: Защита пасеки»;
  2. Запуск компилятора — описывает, как работать с разделом компилятора;
  3. Результат работы компилятора — про то, как получить результат процесса компиляции;
  4. Экспорт схемы в формат игры «Берлога: Защита пасеки» — описывает основные действия для экспорта схемы в игру «Берлога: Защита пасеки».
