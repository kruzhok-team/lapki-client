03. Таблица с устройствами

В центральной части вкладки загрузчика расположена таблица с устройствами, которые можно добавить через список устройств, либо через адресную книгу. Эта таблица состоит из следующих столбцов:

  1. Наименование: название устройства (если это плата МС-ТЮК, то название берётся из адресной книги);
  2. Тип: вид устройства (пусто, если это не плата МС-ТЮК);
  3. Тип: адрес МС-ТЮК (пусто, для других типов устройств);
  4. Что прошиваем: по-умолчанию состоит из двух элементов - выпадающего списка, в котором можно выбрать машину состояний, и кнопки для выбора файла с прошивкой. При выборе файла откроется диалоговое меню через которе нужно выбрать нужный Вам файл, после чего выпадающий список заменится на название файла. Чтобы отменить выбор машины состояний или файла нужно нажать на появивщийся (после выбора) сбоку крестик.

Выбор машин состояний, ограничен типом соответствующего устройства, например в Arduino Uno нельзя загрузить прошивку, скомпилированную для КиберМишки. Если тип устройства неизвестен, то IDE позволит выбрать любую машину состояний, но будьте осторожны, неизвестно, что произойдёт, если Вы загрузите прошивку в непредназначенное для неё устройство.

Напротив каждой записи таблицы находится окошко для галочки. Если запись отмечена галочкой, то значит, над ней будут проводиться операции, такие как загрузка прошивки. Чтобы поставить, либо убрать галочку, достаточно просто кликнуть по области окошка. Вверху, напротив заголовка таблицы, находится окошко для галочки, которое отмечает все устройства, если поставить галочку, либо снимает все отметки, если снять галочку.
