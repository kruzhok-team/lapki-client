00. О среде разработки

Данная среда разработки предоставляет инструменты для работы с киберфизическими системами в парадигме программирования расширенных иерархических машин состояния (ПРИМС).

Далее в тексте, будет употребляться сокращение IDE (англ. integrated development environment — интегрированная среда разработки) для обращения к данной программе.

С IDE можно создавать и редактирвоать схемы машин состояний, компилировать их, а также загружать их на физические устройства.

Помимо работы с физическими устройствами, IDE предоставляет возможность работы со схемами из игры «Берлога: Защита пасеки». Из этой игры можно загрузить схему и отредактировать её здесь, либо можно создать схему с нуля в IDE и затем загрузить её в «Берлога: Защита пасеки».

Разделы руководства, которое Вы сейчас читаете помогут Вам освоиться в IDE. Познакомимся с ними:

  • 00. О среде разработки — Вы сейчас читаете этот раздел.
  • 01. ПРИМС — краткое введение в программирование расширенных иерархических машин состояния;
  • 02. Быстрый старт — пошаговое введение в основы работы с IDE на примерах КиберМишки и МС-ТЮК;
  • 03. Интерфейс среды — подробное описание интерфейса и функциональности IDE;
  • 04. Дополнительные материалы — рекомендовано разработчиками для изучения таких тем, как инженерия, программирование, электроника.

Советы по взаимодействию с руководством:

  • Нажмите кнопку Содержание, чтобы вернуться к меню выбора страниц руководства;
  • Нажмите кнопку Просмотр, чтобы вернуться из раздела Содержание к последней открытой странице;
  • Используйте кнопки Назад и Вперёд для последовательного перемещения по руководству;
  • Нажмите кнопку F1 на вашей клавиатуре для того, чтобы открыть, либо закрыть руководство;
  • Нажмите на кнопку руководства пользователя на боковой панели, чтобы закрыть, либо открыть руководство;
  • При наведении на левый край области руководства у Вас изменится указатель мыши, после чего, Вы можете зажать левую кнопку мыши и сдвинуть её влево (вправо), чтобы растянуть (сжать) область руководства по Вашему усмотрению.

Спасибо, что пользуетесь нашей IDE!
